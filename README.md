# Smart Camera

Smart Camera allows the user to take a photo or video and detect :
- Face
- Text in video
- Text in image

Smart Camera hase modes like :
- Night mode(LAB)
- Black mode(GRAY)
- Fire mode(HSV)
- Normal mode(BGR)
